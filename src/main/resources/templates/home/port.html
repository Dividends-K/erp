About My Journey Into Programming

Originally, I graduated in Media and Printing and Accounting majors. However, life is strange and led me to have the first contact with Programming in 2019.

In 2019, my director requested me to create an effective voucher system to ready our company for making transactions with a strict customer.

Abiding by his order, I devised a simple voucher scheme, which was the first step of my journey to programming. At first, I created a complex MS Excel file to accommodate the scheme. It was met with slow speed and scalability which required resetting to another file each 100 orders.

Eventually, in 2020 I first touched upon programming by developing my first sales-monitoring application using MS Access and VBA, which was not exactly a programming language itself, even though it helped me effectively managed the sales data and raised the firm's revenue by at least 15%.

In 2022, I decided to migrate the back end of my application to MySQL with the purpose of making my application becoming more portable. I completed the migrating to a spare server machine: HPE Proliant DL60 Gen9. However, the process of setting up ODBC on each machine that wanted access to the database made me feel unsatisfied.

In Feb 2023, with the familiarity to SQL and all those above, I decided to learn JAVA with the goal of making an ultra-portable web application that could house my growing database. Besides, my application had always been used for sales-monitoring, I wanted to upgrade it into a full inventory-monitoring application, or even an ERP.

Now in July 2023, the application was partially done, reaching the functionality of the old MS Access one, though having a tighter control on inventory counts.

2. WebNovel Content-Copying Project

My hobby is to read web novel in my kindle which does not have much support for web browsing.

In order to avoid manually copying the contents of those free novels chapter-by-chapter, I developed a piece of API using libraries of Jsoup and Apache POI that helps me parse the links and download the novel into a docx file. The formatting uses Bookerly font family, which was recommended from KindleAmazon for reading, accompanied by proper indentation and chapter references.
