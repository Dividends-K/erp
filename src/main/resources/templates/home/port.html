About My Journey Into Programming

Originally, I graduated with two degrees in Media & Printing, and Accounting majors. However, as if it was my calling, I was led into first contact with Programming in 2020.

At the time, I was requested to create an effective voucher system to ready our company for making transactions with a customer who had a flair for rigidity.

Abiding by his demand, I devised a simple voucher scheme, which was the first step of my journey to programming. At first, I created a complex MS Excel file to accommodate the scheme. It was met with slow speed and scalability which required resetting to another file every 100 orders.

Images from one of the primitive MS Excel books.

Eventually, in 2020, I first touched upon programming by developing my first sales-monitoring application using MS Access and VBA, which was not exactly a programming language itself. The application helped me effectively managed the sales data and raised the firm's revenue by at least 15%.

Some images of the MS Access application

In 2022, I decided to migrate the back end of my application to MySQL, aiming to make my application more portable, to a spare server machine: HPE Proliant DL60 Gen9. However, the process of setting up ODBC on each machine that wanted access to the database was still somewhat underwhelming.

In February 2023, with the familiarity to SQL and all those above, I decided to learn JAVA with the goal of making an ultra-portable web application that could house my growing database. Besides, since my application had always been used for sales-monitoring, upgrading it into a full inventory-monitoring application, or even an ERP, was a natural development.

In September 2023, the application was partially done, reaching the functionality of the old MS Access one, though having a much tighter control on inventory input and output.

A more detailed description can be accessed in the readme.md file if you visit my GitHub at the Project section below.

Every line of codes from my Spring Boot application is condensed into above reports. The Sale page is...a little clustered, but provides clear and uninterrupted views of sales data.

2. WebNovel Content-Copying Project
